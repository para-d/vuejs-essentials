Vue CLI

Installation

npm install -g @vue/cli

Creating a Vue project

vue create <project-name>

Then it will prompt to select features and take you through the other steps

Directory Structure

            /public/            | contains the index file, and the compiled js will be injected to it.
            /src/               | where you can find the source codes
                /assets/
                /components/    | Sub/shared components that are not connected to router
                /router/
                    index.js    | Vue Router configuration
                /views/         | Main view templates that are connected to router
                    Home.vue    | Some default page
                    About.vue   | Some default page
                App.vue         | main component of the app
                main.js         | the main js file. the vue app is created and mounted here.

Running the scripts listed in package.json

npm run <script-name>
