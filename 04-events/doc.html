Events

Long syntax

            <button v-on:click="/*do something here*/">Click</button>

Short syntax

            <button @click="/*do something here*/">Click</button>

Example

Styles

            <style media="screen">
                div.box{ width:100px; height:100px;}
                div.box.c1{ background-color:yellow; }
                div.box.c2{ background-color:red; }
                div.box.c3{ background-color:blue; }
                div.box.c4{ background-color:green; }
                [v-cloak] { display: :none; }
            </style>

Body

            <div id="app" v-cloak>
                <div v-if="visible1" class="box c1">if: visible1: {{visible1}} | visible2: {{visible2}}</div>
                <div v-else-if="visible2" class="box c2">else if: visible1: {{visible1}} | visible2: {{visible2}}</div>
                <div v-else class="box c3">else: visible1: {{visible1}} | visible2: {{visible2}}</div>

                <div v-show="show" class="box c4">show</div>
                <div>
                    <button v-on:click="visible1=!visible1">visible1 toggle</button>
                    <button v-on:click="visible2=!visible2">visible2 toggle</button>
                    <button v-on:click="show=!show">show toggle</button>
                </div>
            </div>
            <script type="text/javascript">
                let app = Vue.createApp({
                    data : function(){
                        return {
                            visible1:false,
                            visible2:false,
                            show:true
                        };
                    }
                });

                app.mount('#app');
            </script>

Event Modifiers

Different events can have unique modifiers. There are some modifiers that are commonly used as well.

Common modifiers

.stop

Stop propagation: Stop the event from propagating into the child nodes.

                    <button @click.stop="...">Click here</button>
.prevent

Prevent Default: Similar to event.preventDefault().

                    <button @click.prevent="...">Click here</button>
.self
.capture
.once
.passive

Unique modifier examples

            <button @click.right="...">Do a right click</button>
            <input @keyup.13="..." /> // enter key pressed

Use cases

Basic behaviour

            <button @click="..."></button>

Modifier chaining

            <a @click.right.prevent.self=""></a>

Just the modifier

            <form @submit.prevent></form>

$event/event variable

            <button @click="do_something_with('some-argument',$event)"></button>
            <button @click="(event) => do_something_with('some-argument',event)"></button>
