Render Control

v-cloak

Helps to keep the templates visible until the application is ready.

v-if, v-if-else, v-else

Decides whether an element should be rendered or not.

v-show

Decides if an element should be set to visible or not. Controlled through CSS display:none

Example

Styles

            <style media="screen">
                div.box{ width:100px; height:100px;}
                div.box.c1{ background-color:yellow; }
                div.box.c2{ background-color:red; }
                div.box.c3{ background-color:blue; }
                div.box.c4{ background-color:green; }
                [v-cloak] { display: :none; }
            </style>

Body

            <div id="app" v-cloak>
                <div v-if="visible1" class="box c1"></div>
                <div v-else-if="visible2" class="box c2"></div>
                <div v-else class="box c3"></div>

                <div v-show="show" class="box c4"></div>
            </div>
            <script type="text/javascript">
                let app = Vue.createApp({
                    data : function(){
                        return {
                            visible1:false,
                            visible2:false,
                            show:false
                        };
                    }
                });

                app.mount('#app');
            </script>
