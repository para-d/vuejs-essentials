Placeholder

Vue uses a custom non-standard placeholder tag in the HTML document so that it can replace it with the real template when it is rendered.

            <div id="app" v-cloak>
                <component1 />
            </div>

Defining the New Component

The custom non-standard Vue components that you use as placeholders, have to be defined after the Vue.createApp line and before the app is mounted.

            <script type="text/javascript">
                let app = Vue.createApp({});

                app.component('component1',{
                    template: `
                        <div><h1>This is Component1</h1></div>
                    `
                });

                app.mount('#app');
            </script>
