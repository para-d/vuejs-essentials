Lifecycle Hooks

You can create the lifecycle hook functions in your options object.

Use Cases
  • Check for authorization
  • API calls
  • Create/remove events
  • Set and clean-up data
            <script type="text/javascript">
                let app = Vue.createApp({
                    created:function(){ /*...*/ },
                    mounted:function(){ /*...*/ },
                    unmounted:function(){ /*...*/ }
                });

                app.component('component-name',{
                    template:`<div></div>`,
                    beforeUpdate:function(){},
                    updated:function(){}
                });
            </script>
